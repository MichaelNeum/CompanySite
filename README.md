 # Company Site — Starter

Simple starter template containing:

- `index.html` — the main HTML file
- `css/styles.css` — basic styles and responsive nav
- `js/app.js` — small JS for year and menu toggle

How to use:

1. Open `index.html` in your browser (double-click or serve with a static server).
2. Edit the files in `css/` and `js/` to customize the site.

Optional quick preview using Python's http.server:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
